# Opening Flow

The opening flow is the core interaction of Stockpackz — a full-screen cinematic sequence that culminates in revealing a tokenized stock. It is designed to feel like unboxing an Apple product mixed with a Pokémon booster pack: premium, minimal, no slot machine.

**Component:** `src/components/capsules/open-capsule-flow.tsx`

## Entry Points

| Trigger | Behavior |
|---------|----------|
| Hero **Open Your First Pack** button | Opens flow in `selecting` phase — user picks pack type |
| Pack card click | Starts the cinematic directly with that pack |
| Final CTA button | Same as hero |

If the wallet is not connected, the flow shows the `connect` phase first (`ConnectWalletPrompt`).

## Phase Timeline

The entire sequence is driven by one deterministic timeline started in `startOpening()`. The winning pull is drawn **up front** with `pickRandomPull()`, so no state is decided mid-animation.

```
idle → connect? → selecting? → focus → rotating → tearing → burst → reveal → swap → result
```

| Phase | Time | What happens |
|-------|------|--------------|
| `focus` | 0 – 0.85s | Pack lifts from below and scales up; backdrop deepens to near-black and blur doubles (camera zoom) |
| `rotating` | 0.85 – 3.05s | Pack sways in 3D perspective; metallic sheen band sweeps the foil; placeholder crinkle sounds |
| `tearing` | 3.05 – 4.25s | Green tear line rips across the seal with escaping glow; pack shudders; smoke particles rise; six mini stock logos fly upward |
| `burst` | 4.25 – 5.75s | White-green radial flash; pack blows out; winning logo appears spinning fast and motion-blurred (identity hidden) |
| `reveal` | 5.75 – 8.1s | Spin decelerates on an ease-out curve; blur clears; massive uppercase company name (up to 4.5rem); rarity label (gold legendary / violet epic) |
| `swap` | 8.1 – 9.2s | Settlement pipeline animates in under one second: `USDG → Uniswap v4 → {TICKER} → Wallet`, ending with "Settled on-chain" |
| `result` | 9.2s+ | Portfolio card: "Added to your portfolio", token amount, USD value, actions |

## Performance (60fps)

- Every animation uses **transform and opacity only** — no layout or paint properties in loops
- 3D rotation via `rotateY` with `transformStyle: preserve-3d` and container `perspective`
- Timers are tracked in a ref and cleared on unmount / close / replay
- Framer Motion springs and cubic-bezier `[0.22, 1, 0.36, 1]` easing throughout

## Crinkle Sound (placeholder)

`playCrinkle()` generates short bandpass-filtered white-noise bursts with the Web Audio API — no audio files shipped. During `rotating`, a 420ms interval plays soft crinkles; a louder one fires at the seal tear. Replace with real foil samples for production. Fails silently when audio is unavailable.

## Confetti

**Legendary only.** `fireLegendaryConfetti()` uses [canvas-confetti](https://github.com/catdad/canvas-confetti) with a gold/white palette and `disableForReducedMotion: true`. No confetti for epic and below.

## Pull Generation

```typescript
const pull = pickRandomPull(capsule);
// Returns: { stock, tokenAmount, valueUsd }
```

- Stock selected by rarity weight (50/30/15/5)
- Fraction generated by `generatePullAmount()` in `tokenized-stocks.ts`
- Result committed to state before the animation begins

## Accessibility

- Close button available in `connect`, `selecting`, and `result` phases; backdrop click dismisses in those phases only (never mid-cinematic)
- Decorative layers are `aria-hidden`
- Confetti respects reduced-motion preferences

## Production Changes

Replace the mock pull with on-chain settlement:

1. User confirms open → sign transaction
2. Pack contract selects stock + amount via verifiable randomness (VRF)
3. Contract swaps USDG for the stock token via Uniswap v4 and transfers to the user
4. Listen for the settlement event; feed the confirmed result into `startOpening()`'s timeline
5. Hook point: `onPullComplete` callback in `page.tsx`
